<h3>Használata:</h3>

- Először is telepítsd az electron.js-t:
```
npm i electron --save-dev
```
- Illetve a kötvetkezőket:
```
npm i path axios xml2js os unzipper
```
- Utána, ha ezzel megvagy, akkor:
```
npm run build
npm run start
```
